원장님을 위한
클로드 코워크 시작하기

클로드 코워크(Cowork)는 클로드가 원장님 대신 실제로 손발이 되어 일을 처리해주는, 요즘 가장 강력한 AI 도구예요. 다만 이걸 제대로 쓰려면 먼저 내 컴퓨터에 클로드를 설치해야 하는데, 윈도우에서 설치가 막막하다는 원장님이 많으셨어요. 그래서 화면 그대로, 한 단계씩 따라만 하면 되도록 자세히 정리했습니다. 5~10분이면 끝나요.

2026년 7월 최신 기준 Windows 10 · 11 WSL 없이 설치
최근에 바뀐 점

이제 채팅과 코워크가 한 화면으로 합쳐졌어요

2026년 7월, 클로드 앱에서 채팅(Chat)과 코워크(Cowork)가 하나의 홈 화면으로 통합됐어요. 코워크 작업이 클라우드에서도 돌아가서 노트북을 닫아도 백그라운드로 계속 진행되고, 휴대폰으로 결과를 확인할 수 있습니다. 이 가이드는 그 최신 화면 기준으로 안내드려요.

먼저, 이것만 준비돼 있으면 돼요

설치 자체는 몇 분이면 끝나요. 다만 두 가지는 꼭 확인해 주세요.

클로드 유료 플랜
Pro·Max·Team 중 하나. 무료 플랜에서는 클로드 코드를 쓸 수 없어요.
윈도우 10 또는 11
Windows 10 (1809 이상) / 11. RAM 4GB 이상.
인터넷 연결
설치와 로그인에 필요해요.
Git for Windows (선택)
있으면 더 많은 명령을 쓸 수 있어요. 없어도 설치·실행은 됩니다.

설치 방법은 두 가지예요

둘 중 편한 쪽으로 하나만 하시면 돼요. 터미널이 낯설면 방법 A를 추천드려요.

A 데스크탑 앱

클로드 앱으로 설치하기

클릭만으로 설치. 검은 화면(터미널) 없이, 클로드 앱 안의 Code 탭에서 바로 사용해요.

터미널이 처음인 분께 추천 B 터미널(PowerShell)

명령어 한 줄로 설치하기

PowerShell에 명령어 한 줄을 붙여넣어 설치해요. claude 명령을 터미널에서 직접 쓰고 싶은 분께.
방법 A

클로드 앱으로 설치하기

터미널을 열 필요가 없어요. 카카오톡 설치하듯 앱을 깔고, 앱 안에서 클로드 코드를 씁니다.

1

클로드에 가입하고 유료 플랜을 준비해요

claude.ai에 접속해 Continue with Google로 가입하세요. 클로드 코드는 유료 플랜(Pro 이상)이 필요해요. 무료 플랜에서는 이 기능이 안 열립니다.

여기를 눌러주세요!

claude.ai 첫 화면 — Continue with Google을 누르면 가입이 시작돼요.

2

윈도우용 데스크탑 앱을 내려받아요

claude.ai/download에 접속하면 내 컴퓨터에 맞는 버전(Windows)이 자동으로 안내돼요. 가운데 검은 버튼을 누르면 설치 파일이 받아집니다.

여기를 눌러주세요!

claude.ai/download 화면 — 검은 버튼을 누르면 윈도우용 설치 파일이 받아져요.

3

설치 파일을 실행하고 로그인해요

다운로드 폴더의 설치 파일(예: Claude-Setup.exe)을 더블클릭하면 설치가 진행돼요. 끝나면 시작 메뉴에서 Claude를 실행하고, 1단계에서 가입한 계정으로 로그인하세요.

Git이 필요할 수 있어요 윈도우에서 내 컴퓨터의 파일을 직접 다루는 로컬(Local) 세션을 쓰려면 Git for Windows가 설치돼 있어야 해요. 없다면 지금 함께 설치해 두면 편합니다.
4

상단의 세 가지 탭을 알아둬요

앱 위쪽 가운데에 Chat · Cowork · Code 탭이 있어요. 우리가 쓸 클로드 코드는 「Code」 탭이에요. 그 옆의 채팅과 코워크는 최근 한 홈 화면으로 합쳐지고 있는 중이고요.

Claude
Chat CoworkNEW Code
세션
· my-project
· 새 세션 +
Local · C:\Users\hyunji\my-project
무엇을 도와드릴까요? 폴더를 고르고 할 일을 적어주세요.
할 일을 입력하세요… ↑

데스크탑 앱 재현 화면 — 상단 Code 탭이 클로드 코드예요.

5

첫 세션을 시작해요

Code 탭에서 순서대로 눌러주세요.

① Local(내 컴퓨터) 선택 → ② Select folder로 작업할 폴더 선택 → ③ 보낼 곳 옆에서 모델 선택 → ④ 할 일을 입력 (예: "이 폴더에 CLAUDE.md를 만들어줘") → ⑤ 클로드가 제안한 변경사항을 초록/빨강 diff로 확인하고 Accept로 승인!

Claude — Code
세션
· my-project
Local · 모델: Claude Opus
CLAUDE.md 파일을 만들었어요. 아래 변경사항을 확인해 주세요.
CLAUDE.md +3 -0 + # 프로젝트 안내 + 이 폴더는 크래빗 실습용입니다.
Accept 를 눌러 변경을 적용하세요 ↑

기본은 수동 모드라, 내가 승인(Accept)하기 전까지 파일은 바뀌지 않아요.

방법 B

터미널(PowerShell)로 설치하기

명령어 한 줄이면 끝나요. 예전과 달리 Node.js도, WSL도 필요 없어요. 단일 실행파일을 내려받아 자동으로 설치되고, 이후 업데이트도 알아서 됩니다.

1

PowerShell을 열어요

윈도우 시작 버튼을 누르고 PowerShell이라고 검색한 뒤, Windows PowerShell을 클릭해 실행하세요. 관리자 권한은 필요 없어요.

PowerShell이 맞는지 확인하는 법 검은(또는 파란) 창의 맨 앞이 PS C:\Users\...> 처럼 PS로 시작하면 PowerShell이 맞아요. C:\Users\...> 처럼 PS가 없으면 그건 CMD예요(방법이 조금 달라요 — 아래 문제 해결 참고).
2

설치 명령어를 붙여넣고 Enter를 눌러요

아래 명령어를 복사해서 PowerShell 창에 붙여넣고(마우스 오른쪽 클릭 = 붙여넣기) Enter를 누르세요.

PS> irm https://claude.ai/install.ps1 | iex

그러면 아래처럼 설치가 진행돼요.

Windows PowerShell
PS C:\Users\hyunji> irm https://claude.ai/install.ps1 | iex Downloading Claude Code… Installing to C:\Users\hyunji\.local\bin Adding to PATH… ✓ Claude Code installed successfully! Restart your terminal, then run claude to start. PS C:\Users\hyunji>

설치가 끝나면 ✓ installed successfully 메시지가 떠요.

3

PowerShell을 완전히 닫았다가 다시 열어요

방금 설치가 PATH(윈도우가 프로그램을 찾는 경로)에 등록됐는데, 이건 창을 새로 열어야 반영돼요. 열려 있던 PowerShell 창을 닫고 새 PowerShell 창을 다시 여세요.

4

제대로 설치됐는지 확인해요

새 창에 아래를 입력하고 Enter를 누르세요.

PS> claude --version

2.1.211 (Claude Code) 처럼 버전 번호가 나오면 성공이에요.

혹시 "claude is not recognized..." 라고 나온다면? 3단계처럼 창을 아직 새로 안 연 경우예요. 창을 완전히 닫고 다시 열어보세요. 그래도 안 되면 claude doctor를 입력해 무엇이 문제인지 진단할 수 있어요.
5

클로드 코드를 실행하고 로그인해요

작업할 폴더에서 아래를 입력하면 클로드 코드가 시작돼요.

PS> claude

처음 실행하면 브라우저가 자동으로 열리며 로그인을 안내해요. 가입한 클로드 계정으로 로그인하면, 아래와 같은 환영 화면이 나타납니다.

claude
✻ Welcome to Claude Code
/help 로 도움말 · /status 로 현재 설정 확인
cwd: C:\Users\hyunji\my-project
> 무엇을 도와드릴까요? 여기에 입력하세요…
? 를 누르면 단축키 · Enter 로 전송

클로드 코드 첫 실행 화면 — 아래 입력창에 할 일을 적고 Enter를 누르면 시작돼요.

6

(권장) Git for Windows도 설치해요

Git for Windows가 있으면 클로드 코드가 Bash 도구까지 쓸 수 있어 더 많은 일을 할 수 있어요. 없어도 동작하지만, 있으면 훨씬 편합니다.

winget으로 설치하는 방법도 있어요 명령어 설치가 막히면 winget install Anthropic.ClaudeCode 로도 설치할 수 있어요. (이 방식은 자동 업데이트가 안 되니, 가끔 winget upgrade Anthropic.ClaudeCode로 최신화해 주세요.)

채팅 · 코워크 · 코드, 뭐가 다른가요?

같은 클로드지만 하는 일이 달라요. 헷갈리기 쉬워서 정리해 드려요.

채팅 Chat

GPT처럼 대화만 하는 곳이에요. 내 컴퓨터 파일에는 손대지 않아요.

코워크 Cowork

클로드가 알아서 백그라운드로 일해주는 자동 에이전트예요. 최근 채팅과 한 화면으로 합쳐졌고, 클라우드에서 돌아 노트북을 닫아도 계속돼요.

코드 Code

바로 이게 클로드 코드예요. 내 컴퓨터의 파일을 직접 열고 고치며, 변경은 내가 하나씩 확인·승인해요.

자주 막히는 부분

이 세 가지만 알면 대부분 해결돼요.

'irm' is not recognized...

지금 창이 CMD예요(PowerShell 아님). 시작 메뉴에서 PowerShell을 검색해 다시 열고 명령어를 붙여넣으세요. 프롬프트 앞이 PS로 시작하는지 확인!

claude is not recognized...

설치는 됐지만 창을 아직 새로 안 연 경우예요. PowerShell 창을 완전히 닫고 다시 열어 주세요. 그래도 안 되면 claude doctor로 진단해 보세요.

로그인하니 "요금제를 올려달라(upgrade)"고 나와요

무료 플랜에서는 클로드 코드를 쓸 수 없어요. Pro·Max·Team 중 하나로 구독한 계정으로 로그인해야 합니다.

자주 묻는 질문

방법 A와 방법 B, 둘 다 해야 하나요?

아니요. 둘 중 하나만 하면 돼요. 터미널이 낯설면 방법 A(앱), 명령어로 직접 다루고 싶으면 방법 B(터미널)를 고르세요. 나중에 둘 다 써도 설정은 서로 공유돼요.

예전에는 WSL이나 Node.js가 필요하다던데요?

이제는 윈도우에서 그냥 설치돼요. 네이티브 설치 프로그램이 단일 실행파일을 내려받아 주기 때문에 WSL도, Node.js도 따로 안 깔아도 됩니다.

무료로 쓸 수는 없나요?

클로드 코드는 유료 플랜(Pro 이상) 전용이에요. 무료 클로드 계정으로는 대화(Chat)만 가능합니다.

설치한 뒤 업데이트는 어떻게 하나요?

네이티브(방법 B의 기본) 설치는 자동으로 최신화돼요. 바로 올리고 싶으면 claude update를 입력하세요. winget으로 설치했다면 winget upgrade Anthropic.ClaudeCode를 가끔 실행해 주세요.

설치하시다 막히시면 편하게 연락 주세요. 화면 공유로도 도와드릴게요.
