원장님을 위한
클로드 코워크 시작하기

클로드 코워크(Cowork)는 원장님 대신 컴퓨터로 일을 해주는 AI 비서예요. 출석부 정리, 학부모 안내문 쓰기, 반복되는 업무 자동화까지 대신 해줍니다. 그런데 이걸 쓰려면 컴퓨터에 클로드를 한 번 설치해야 하는데, 이 설치 부분이 막막하다는 원장님이 많으셨어요. 그래서 맥이든 윈도우든, 어려운 말 없이 사진 그대로 순서대로 누르기만 하면 되도록 정리했어요.

잠깐, 이건 알고 가면 좋아요

'설치'가 뭔가 어렵게 느껴지시죠? 딱 카카오톡 깔듯이 하면 돼요

인터넷 창에 주소를 쳐서 쓰는 건 '웹'이에요. 웹에서는 클로드와 이야기만 할 수 있어요. 반면 내 컴퓨터에 프로그램을 설치하면(카카오톡처럼요), 클로드가 내 컴퓨터의 파일과 폴더를 직접 열고 정리해줄 수 있어요. 이게 바로 코워크예요.

Node.js니 파워셸이니 하는 어려운 건 하나도 몰라도 됩니다. 아래 순서만 따라오세요.

시작 전, 이 3가지만요

복잡한 준비물은 없어요. 딱 세 가지만 확인해 주세요.

1
노트북 (맥 또는 윈도우)
우리가 늘 쓰는 그 컴퓨터면 돼요. 맥·윈도우 모두 가능해요. 맥은 macOS 11 이상이면 되고, 윈도우는 설치 오류가 나면 윈도우 버전을 업데이트한 뒤 다시 시도해 보시는 걸 추천해요.
2
클로드 유료 플랜
코워크는 유료 기능이에요. 가장 저렴한 Pro 플랜이면 충분해요. (무료로는 안 돼요)
3
인터넷 연결
설치할 때만 필요해요.

이제 실제 화면 그대로 따라오세요

아래 실제 영상은 윈도우 화면 예시예요. 맥도 흐름은 똑같고, 화면이 달라지는 부분만 각 단계에서 맥은 이렇게로 따로 짚어드릴게요. 눌러야 하는 곳은 분홍 동그라미로 표시해 드릴게요.

1

클로드에 가입해요

claude.ai에 들어가서 Continue with Google 버튼을 누르면, 늘 쓰시는 구글 계정으로 1분 만에 가입돼요.

여기를 눌러주세요!

claude.ai 첫 화면 — Continue with Google을 누르면 가입이 시작돼요.

2

유료 플랜(Pro)을 신청해요

코워크는 유료 기능이라, 로그인한 뒤 Upgrade(업그레이드)에서 가장 저렴한 Pro 플랜을 신청해 주세요. 카드 결제 한 번이면 끝나요.

왜 유료인가요? 무료 계정으로는 클로드와 대화만 가능하고, 컴퓨터 일을 대신 시키는 '코워크'는 열리지 않아요.
3

내 컴퓨터에 맞는 앱을 내려받아 설치해요

claude.ai/download에 들어가면 내 컴퓨터에 맞는 버튼이 자동으로 보여요. 윈도우는 Download for Windows를 눌러 받은 파일을 두 번 클릭하고, "이 앱이 디바이스를 변경할 수 있도록 허용하시겠어요?"가 뜨면 예를 누르면 설치가 저절로 진행돼요.

맥(MacBook)은 이렇게 Download for Mac을 누르면 설치 파일(.dmg)이 받아져요. 그 파일을 두 번 클릭한 뒤, 나오는 창에서 Claude 아이콘을 Applications(응용 프로그램) 폴더로 드래그하면 설치 끝이에요. 처음 열 때 "인터넷에서 받은 앱을 여시겠어요?" 물음이 나오면 열기를 눌러주세요. (칩이 M1~M4든 인텔이든 같은 파일이라 그냥 받으시면 돼요.)

위 영상은 윈도우 설치 예시예요. 맥은 받은 .dmg 파일을 열어 아이콘을 응용 프로그램 폴더로 드래그하는 것만 달라요.

4

앱이 열리면 로그인해요

설치가 끝나면 클로드 앱이 저절로 열려요. 시작하기 → Google로 계속하기 → 내 구글 계정 선택, 이 세 번만 누르면 로그인 끝!

1번에서 가입한 구글 계정을 고르면 돼요. 중간에 "허용할까요?" 물음이 나오면 모두 예/허용을 누르세요.

앱이 안 열렸다면? 윈도우는 화면 왼쪽 아래 시작 버튼(윈도우 로고)에서, 맥은 Launchpad나 Spotlight(⌘+스페이스바)에서 Claude를 검색해 열면 돼요.
5

이제 일을 시켜보세요 — 이게 코워크예요

로그인하면 바로 보이는 홈 화면, 여기가 채팅과 코워크가 합쳐진 새 홈이에요. 가운데 칸에 시킬 일을 말하듯이 그냥 적고, 주황색 보내기 버튼을 누르면 클로드가 알아서 일을 시작해요.

  • "이 엑셀 출석부에서 이번 달 3번 이상 결석한 학생만 뽑아줘"
  • "학부모님께 보낼 이번 주 안내문 초안 써줘"

일을 적고 보내면, 오른쪽에 진행 상황 창이 뜨면서 클로드가 알아서 작업해요. 결과가 마음에 안 들면 또 말로 고쳐달라고 하면 돼요.

화면이 조금 다르게 보여도 괜찮아요 클로드는 업데이트가 잦아 버튼 위치가 조금씩 바뀔 수 있어요. 흐름은 똑같으니 그대로 따라 하시면 돼요.

코워크, 웹으로 쓸까 데스크탑 앱으로 쓸까?

클로드 코워크는 ① 웹사이트(claude.ai)에서 바로 접속하거나 ② 데스크탑 앱을 설치해서 쓸 수 있어요. 둘 다 코워크를 쓸 수 있지만, 할 수 있는 일의 범위가 다릅니다.

데스크탑 앱 권장
  • 내 컴퓨터의 파일을 직접 열고 정리·작성할 수 있음
  • 화면 속 다른 프로그램을 대신 조작(컴퓨터 사용)하거나 브라우저를 직접 다뤄줌
  • 강의에서 실습하는 대부분의 기능이 여기서 작동
웹 접속
  • 설치 없이 브라우저만 있으면 바로 사용
  • 리서치, 문서 작성, 연결된 서비스(노션·슬랙 등) 작업은 문제없이 가능
  • 단, 내 컴퓨터 파일을 직접 만지거나 다른 프로그램을 조작하는 기능은 안 됨

강의 실습을 온전히 따라오시려면 데스크탑 앱 설치를 권장드려요.

윈도우는 버전에 따라 앱 사용이 어려울 수 있어요 (맥은 해당 없음)

맥(MacBook) 사용자라면 이 부분은 건너뛰셔도 돼요 — 맥은 버전 제약 없이 그대로 쓰시면 됩니다. 데스크탑 앱의 '컴퓨터 사용' 기능은 윈도우에서만 특정 기능(Hyper-V)이 필요한데, 이 기능이 윈도우 Home 에디션에는 들어있지 않아요.

내 컴퓨터 버전앱 설치 · 사용
윈도우 Pro / Enterprise / Education 문제없음
윈도우 Home 컴퓨터 사용 기능이 제한될 수 있음
맥(Mac) 문제없음 (macOS 11 이상)

내 윈도우 버전은 설정 > 시스템 > 정보의 'Windows 사양 > 에디션' 항목에서 확인할 수 있어요.

윈도우 Home이라서 앱 사용이 안된다면?

데스크탑 앱이 어려운 경우, claude.ai 웹사이트로 접속하면 코워크를 그대로 사용할 수 있어요. 설치가 필요 없어요.

단, 웹 접속 시 아래 기능은 사용할 수 없어요.

  • 내 컴퓨터의 로컬 파일 직접 열기·저장
  • 화면 속 다른 프로그램 대신 조작(컴퓨터 사용)
  • 브라우저 직접 제어
  • 라이브 아티팩트

즉, 컴퓨터를 직접 조작하는 실습을 제외한 리서치·문서 작업·서비스 연동 실습은 웹으로도 충분히 따라오실 수 있어요.

코워크 협업 에러 해결

원장님! '협업'을 누르면
이렇게 에러가 뜨시나요?

이 화면에서 '협업'을 눌렀더니…
+ 채팅 협업
↓
Missing HCS services:
HNS, vmcompute, vfpext

괜찮아요! 아주 흔한 에러라, 아래 4단계만 그대로 따라 하면 금방 해결돼요.

윈도우의 가상화 기능이 꺼져 있어서 생기는 문제예요. 아래 명령어 한 줄로 그 기능을 켜주면 돼요.

  1. 화면 아래 검색창에 PowerShell을 입력해요

    윈도우 맨 아래 작업표시줄의 검색창(돋보기)에 PowerShell이라고 칩니다.

  2. 그냥 클릭하지 말고, 마우스 오른쪽 버튼으로 클릭해요

    검색 결과에 나온 Windows PowerShell을 오른쪽 버튼으로 클릭(우클릭)하면 메뉴가 뜨는데, 거기서 관리자 권한으로 실행을 눌러요. 왼쪽 클릭으로 그냥 열면 안 돼요 — 꼭 우클릭!

  3. 검은 창이 열리면 아래 명령어를 붙여넣고 Enter

    아래 '복사' 버튼을 누른 뒤, 검은 창에서 마우스 오른쪽 클릭(붙여넣기) → Enter를 누르세요.

    Enable-WindowsOptionalFeature -Online -FeatureName VirtualMachinePlatform -All

  4. "재시작할까요? [Y/N]"이 나오면 Y 또는 바로 Enter

    Y를 입력하거나 그냥 Enter를 누르면 컴퓨터가 재시작돼요.

  5. 재시작 후, 협업이 에러 없이 바로 작동해요!

    컴퓨터가 다시 켜지면 클로드를 열고 '협업'을 눌러보세요. 이제 에러 없이, 프롬프트만 입력하면 코워크가 바로 일을 시작해요.

✓
이제 코워크의 협업(컴퓨터 사용) 기능이 정상 작동해요. 마음껏 시켜보세요!
이제 진짜 시작이에요

코워크로, 학원 일의 절반을 맡겨보세요

설치는 끝났어요. 코워크는 원장님이 매일·매주 반복하는 일을 대신 해주는 데 특히 강해요. 예를 들면요.

✓
출결 · 성적 정리
엑셀 출석부에서 결석·지각 학생을 뽑고, 반별 성적표를 표로 정리
✓
학부모 안내문 · 문자 초안
공지, 상담 안내, 결석 알림 문자까지 우리 학원 말투로 한 번에
✓
상담일지 · 학생 기록 정리
흩어진 메모를 학생별로 모아 깔끔한 상담 기록으로
✓
교재 · 시간표 · 과제 자료 만들기
레벨별 문제지, 주간 시간표, 안내 자료를 원장님 대신 뚝딱
✓
매달 반복되는 일 자동화
"매월 1일 이번 달 학원 리포트 만들어줘"처럼 예약해두면 알아서 해둬요

크래빗 아카데미는 이렇게 학원 운영과 학생 관리를 AI로 자동화하는 법을 원장님들과 함께 하나씩 만들어가고 있어요. 더 많은 자료와 강의도 준비돼 있으니 편하게 둘러보세요.

혹시 이런 게 궁금하다면

로그인했는데 "요금제를 올려달라"고 떠요.

무료 계정이라서 그래요. Pro 플랜을 신청한 계정으로 로그인하면 코워크가 열려요. (2번 단계)

설치가 너무 어렵게 느껴져요.

설치가 잘 안 되시면, 사실 claude.ai(웹)에서도 코워크를 바로 쓸 수 있어요. 앱 설치 없이 인터넷 창에서 시작하는 방법이니, 급하면 이 방법부터 써보셔도 돼요.

중간에 막히면요?

그럴 수 있어요. 아래 크래빗 아카데미 채널로 알려주시면 화면을 함께 보며 도와드릴게요. 혼자 끙끙 앓지 마세요!
